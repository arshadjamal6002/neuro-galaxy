# Audio Files

This directory should contain the following audio files for the immersive sound system:

## Required Files

1. **ambient.mp3** - Ambient background sound that loops continuously
   - Recommended: Space/atmospheric ambient sound
   - Volume: Set to 0.3 in code
   - Should loop seamlessly

2. **click.mp3** - Click/interaction sound effect
   - Recommended: Short, subtle click or beep sound
   - Volume: Set to 0.6 in code
   - Should be brief (0.1-0.5 seconds)

## File Sources

You can find free ambient and sound effect files from:
- [Freesound.org](https://freesound.org)
- [Zapsplat](https://www.zapsplat.com)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## Example Search Terms

- Ambient: "space ambient", "atmospheric", "sci-fi ambient", "deep space"
- Click: "ui click", "button click", "soft beep", "interface sound"

## Note

The sound system will automatically handle autoplay restrictions by waiting for the first user interaction before playing the ambient sound.
